# openiptv

## Overview

openiptv is an open-source IPTV client based on ElectronJS and React that allows users to stream live television channels over the internet. It supports various streaming protocols and provides a user-friendly interface for managing and viewing channels.

## Features

- User-friendly interface
- Cross-platform compatibility
- Live TV, Movies and Series support
- And more...

## Installation

To install openiptv, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/amineflex/openiptv.git
   ```
2. Navigate to the project directory:
   ```bash
   cd openiptv
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Usage

After installation, you can start the application and begin adding your favorite IPTV channels. Use the interface to manage your channel list.
